# Roadmap

Planned milestones. Dates are aspirational; scope may shift. Tracked in more
detail via GitHub milestones and issues.

## v0.1.0 — Foundation ✅

- Core three-lane runner, procedural world, pooling
- Save system, settings, localization
- Progression (achievements, missions, daily rewards, store)
- UI suite, audio, camera juice
- Tests + CI + Android export configuration

## v0.2.0 — Content & Feel

- [x] Movement depth: **double jump** and **dash** (with i-frames)
- [x] Additional power-ups: **Slow Motion** and **Invincibility**
- [x] Apply selected character/skin colour to the runner in-game
- [ ] Real art pass (character models, obstacle variety, environment themes)
- [ ] Audio pass (music tracks, full SFX set)
- [ ] Biome/theme rotation as distance increases
- [ ] Tutorial / first-run onboarding

## v0.3.0 — Progression Depth

- [ ] Character passive abilities and upgrade trees
- [ ] Weekly missions and seasonal events (offline, calendar-driven)
- [ ] Cosmetic trails and board skins
- [ ] Replay of best run (ghost)

## v0.4.0 — Polish & Accessibility

- [ ] Colorblind-friendly palettes
- [ ] Left-handed layout and remappable on-screen buttons
- [ ] Reduced-motion mode
- [ ] Performance profiling pass on low-end reference devices

## v1.0.0 — Play Store Launch

- [ ] Store listing assets (screenshots, feature graphic, trailer)
- [ ] Content rating + data-safety declarations
- [ ] Staged rollout via internal → closed → open testing
- [ ] Crash-free session target ≥ 99.5%

## Backlog / ideas

- Endless mode variants (mirror, hardcore, zen)
- Daily challenge seed shared across devices via a code (no server)
- Cloud save via user-provided storage (opt-in, still no backend)
